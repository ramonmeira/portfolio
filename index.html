Como trabalho

Utilizando a metodologia SCRUM para dividir o projeto em ciclos de atividades com entregas períodicas e incrementais. Dessa forma, é possível:

  • avaliar o projeto conforme ele se desenvolve
  • começar a utilizar o produto mais cedo, mesmo que sem todas as funcionalidade
  • e reavaliar as funcionalidades usando-as

Quais são os próximos passos?

  • Discutir as ideias para o projeto.
  • Negociar a proposta e as entregas.
  • Avaliar o produto e melhora-lo a cada iteração.
